Since you are likely working on a Windows computer for your school labs, here is the easiest way to check if a folder is truly "local."

Method 1: The "Properties" Check (Most Accurate)
This is the best way to see exactly where a folder "lives."

Right-click on the folder you want to use.

Select Properties at the very bottom of the menu.

Look for the Location field on the "General" tab.

It IS Local if: The path starts with a drive letter, usually C: (Example: C:\Users\YourName\Documents).

It is NOT Local if: The path starts with two backslashes \\ (Example: \\SchoolServer\Share). This indicates a network path.

Method 2: The "OneDrive" Warning
Since you are extracting a ZIP file for a cybersecurity tool, you should be careful with OneDrive.

Even though OneDrive folders are technically on your hard drive, they "sync" to the internet. Sometimes, this syncing process locks files, which can cause cybersecurity tools (which often have thousands of tiny files) to crash or fail during extraction.

Safest Bet: Create a folder directly on your C: drive to be sure.

Go to This PC -> Local Disk (C:)

Right-click in the white space -> New -> Folder.

Name it something like CyberLab.

If you put your ZIP file in that C:\CyberLab folder, you are 100% local and safe from sync errors.

An absolute path is just the "full address" of the folder, starting from the drive letter (like C:). It tells the computer exactly where to go without any guessing.

If the instructions ask for the "root directory," they usually mean one of two things. Here is how to figure out which one you need:

1. The Root of the Drive (System Root)
If the tool wants to know which drive to use, the absolute path is simply the drive letter and a backslash.

What to type: C:\

2. The Root of Your Project Folder (Most Likely)
If the tool wants to know where you extracted that ZIP file, it wants the full address of the folder you created (like the CyberLab folder we discussed).

How to get this path easily:

Open your folder in Windows File Explorer.

Click inside the Address Bar at the very top of the window (where it usually shows the folder name).

The text will change to a highlighted code. This is your absolute path.

Copy that text (Ctrl+C).

Example: If you made the folder on your C drive, the absolute path you copy/paste will look like this: C:\CyberLab
