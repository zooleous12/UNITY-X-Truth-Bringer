1. Introduction to Python  
These commands assist with Python installation, script execution, and fundamental Python usage.

Check Python Version    
Alternatively, for Python 3:  
python3 --version  
This command displays the installed Python version on the system.

Execute a Python Script  
python your_script.py  
Or for Python 3:  
python3 your_script.py

Launch Python Interactive Mode  
This opens the Python shell for executing commands interactively:  
python  
Or:  
python3

Exit Python Interactive Shell  
exit()  
Or:  
quit()

Install Python Packages Using pip  
pip install package_name  
Example:  
pip install requests

Upgrade pip (Package Manager)  
pip install --upgrade pip

Uninstall Python Packages  
pip uninstall package_name

List Installed Packages  
pip list

After installing Python, the following commands demonstrate basic script writing and execution.

5. Networking and Troubleshooting  
These Python commands and libraries facilitate network analysis and troubleshooting.

Retrieve IP Address (socket library)  
import socket  
hostname = socket.gethostname()  
ip_address = socket.gethostbyname(hostname)  
print(f"Hostname: {hostname}")  
print(f"IP Address: {ip_address}")

Ping a Server (os library)  
import os  
hostname = "google.com"  
response = os.system(f"ping -c 1 {hostname}")  
if response == 0:  
    print(f"{hostname} is reachable.")  
else:  
    print(f"{hostname} is not reachable.")

Scan Open Ports on a Network (socket library)  
import socket  

def scan_ports(host, ports):  
    open_ports = []  
    for port in ports:  
        with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:  
            s.settimeout(1)  
            if s.connect_ex((host, port)) == 0:  
                open_ports.append(port)  
    return open_ports  

print(scan_ports("127.0.0.1", range(20, 1025)))

Check Internet Speed (speedtest-cli package)  
First, install the package:  
pip install speedtest-cli  
Then, use it in Python:  
import speedtest  
st = speedtest.Speedtest()  
print("Download speed:", st.download())  
print("Upload speed:", st.upload())

Parse Network Packets (scapy library)  
Install Scapy:  
pip install scapy  
Then, in Python:  
from scapy.all import *  
def packet_callback(packet):  
    print(packet.summary())  
sniff(prn=packet_callback, count=10)  # Capture 10 packets

Check Open Connections (psutil library)  
Install psutil:  
pip install psutil  
Then, in Python:  
import psutil  
connections = psutil.net_connections()  
for conn in connections:  
    print(conn)

6. Debugging and Troubleshooting  
These commands assist in debugging Python scripts and resolving common errors.

Set a Debugging Breakpoint  
breakpoint()  # Pauses execution and launches debugger

Print Errors During Execution  
try:  
    x = 10 / 0  
except ZeroDivisionError as e:  
    print(f"Error: {e}")

Log Messages for Debugging (logging library)  
import logging  
logging.basicConfig(level=logging.DEBUG)  
logging.debug("This is a debug message.")  
logging.info("This is an info message.")  
logging.warning("This is a warning message.")  
logging.error("This is an error message.")
